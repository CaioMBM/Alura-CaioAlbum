# Alura-CaioAlbum
Projeto criado durante a Imersão Arquitetura Web com IA (2026) da Alura. É um álbum de figurinhas que coloquei em cada página tem um tópico e as figurinhas são referentes ao tópico da página onde elas estão.

# Hospedagem Remota (frontend e backend)

- Backend: Render
